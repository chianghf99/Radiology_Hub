# Project Rules

## 1. Schedule Parsing Rules (班表讀取與解析規則)
- **醫師休假資訊來源**：醫師休假請統一參考 **NI 班表文件最下方的說明/註記**（例如：`休假：黃俊肇（7/17–7/21）`），**切勿**參考 EVT 班表內部的註記。
- **門住急 MRI 週次對應**：MRI 班表的 `W1` ~ `W5` 欄位固定對應 **「週一至週五」**，而非日曆上的第幾週。
  - `W1` = 週一
  - `W2` = 週二
  - `W3` = 週三
  - `W4` = 週四
  - `W5` = 週五
- **代班資料 (covers) 填寫**：休假期間每一天的代班對應，不論是跨天或單日，皆須在資料庫的 `covers` 中以日期為 Key 逐日完整列出，不可遺漏休假的最後一日。
